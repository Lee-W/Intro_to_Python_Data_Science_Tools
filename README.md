## Usage

### Install dependency
```sh
pip install -r requirements.txt
```

- To use [RISE]()

	```sh
	jupyter-nbextension install rise --py --sys-prefix
	jupyter-nbextension enable rise --py --sys-prefix
	```

### Run
```sh
jupyter-notebook
```
